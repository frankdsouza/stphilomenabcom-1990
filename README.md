# 1990 Commerce Batch — GitHub Pages site

This repo is a Jekyll site for our 1990 Commerce Batch memories.

## To add/edit your profile
1. Create a new file under `_classmates/your-name.md` (example given).
2. Fill in your info, upload a photo in `/assets/images/` if desired.
3. Commit or open a Pull Request.

GitHub Pages builds automatically — site visible at https://1990commercebatch.github.io/
